  ## Project requirements:
    * Apache Maven 3.6.0
    * JDK 11.0.6
    * JUnit 5
    
  ### Installing

First, change your directroy to the project folder.
```
cd Part-1/snack-machine 
```
Second, install all maven dependencies.
```
mvn clean install
```

## Running the tests

To run all testcases :
```
mvn clean test
```
