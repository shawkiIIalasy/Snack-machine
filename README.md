  ## Project requirements:
    * Apache Maven 3.6.0
    * JDK 11.0.6
    * JUnit 5
    
  ## Build:
    * change your directory to project use `cd Part-1/snack-machine`.
    * use `mvn clean install` to install all dependencies.
    * use `mvn clean test` to test all test cases.
